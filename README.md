# 狩歌 Pad 抢词版

这是一个无需构建工具的静态网页项目，适合放在 iPad 浏览器里给 2-4 人线下玩。

## 运行方式

直接打开：

```text
D:\Code\caru-uta-pad\index.html
```

如果浏览器限制本地音频文件，也可以用任意静态服务器托管这个目录。

## 玩法落地

1. 开局选择玩家人数、词卡数量、单轮时长和惩罚模式。
2. 上传或播放一首带日语歌词的歌曲，也可以外放别的设备或现场唱。
3. 听到桌面词卡里的词时，玩家先按自己颜色角。
4. 抢到的人 2 秒内点词卡，桌边玩家共同判定正确或错误。
5. 正确按卡面分得分，连续答对 3 次额外 +2 分。
6. 错误 -1 分，轻惩罚冻结 2 秒，标准惩罚冻结到本首歌结束。
7. 同一首歌里已经拿到的词再次出现时，点下方已拿词筹码，额外 +1。

## 词卡分值

- 高频词：1 分，例如「君、好き、夜、今、夢」
- 中频词：2 分，例如「約束、涙、未来、声」
- 稀有词：3 分，例如「奇跡、孤独、景色」

## 自定义词库

开局页可以展开“编辑词库”，支持两种导入格式。

CSV/TSV：

```text
君,きみ,1
約束,やくそく,2
奇跡,きせき,3
```

JSON：

```json
[
  { "word": "君", "kana": "きみ", "value": 1 },
  { "word": "約束", "kana": "やくそく", "value": 2 }
]
```

导入后的词库会保存在当前浏览器本地。分值会自动限制在 1-3，重复词会只保留第一张。
